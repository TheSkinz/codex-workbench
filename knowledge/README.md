# Public-Safe Knowledge Layer

This layer contains reusable, public-safe industry concepts and synthetic
examples. It is not a private knowledge vault and is never a source of customer
records, facility facts, rates, credentials, or proprietary source documents.

Every knowledge record should identify:

- source authority and public source links;
- confidence and known uncertainty;
- intended audience and scope;
- review trigger or expiry;
- whether it is a concept, pattern, example, or proposal.

If a private canonical system exists elsewhere, publish only a deliberately
sanitized release packet using `docs/pack-release.md`.
