# Workbench Drift

Read `docs/loops/drift.md`. Work in a clean non-`main` worktree. Quote the
current conflicting lines, identify the authoritative source, propose the
smallest correction, and return a review record or draft PR plan. Never edit
`main`, silently rewrite history, or guess at unresolved domain truth.
