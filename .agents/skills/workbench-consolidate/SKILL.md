# Workbench Consolidation

Read `docs/loops/consolidation.md`. Select one clear duplicate, stale section,
or broken reference in the public documentation layer. Preserve source meaning
and history, make one bounded change, run validation, and prepare a reviewable
branch or draft PR. No broad cleanup and no direct `main` mutation.
