# Workbench Review

Read `docs/loops/review.md` and preview the queue. Select one item, quote the
evidence, identify contradictions and uncertainty, and write one review record
only when the run is authorized to write. Never resolve conflicting claims or
apply a human-gated decision without explicit approval.
